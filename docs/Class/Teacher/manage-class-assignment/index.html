Class Management: Assignments

Step 1: Access https://coursemind.io/courses

Step 2: Access a class that you manage.

Step 3: Select [Assignments] on the toolbar.

1. Add an Assignment​

Step 1: Add an assignment​

Press the [Add] button to start adding an assignment.

Step 2: Enter the assignment detail.​

Input the assignment Title, Description, Max Points, Deadline, Topic, State (draft/published), and attached file.

Step 3: Save Assignment​

Press [Save] to create the assignment.

The system will save and display the list of assignments with the created information, along with accompanying actions like View and Edit.

2. Edit Assignment​

Step 1: Select [Edit Assignment]​

Click the [Edit] button at the right corner of the assignment to start editing the assignment.

Step 2: Enter the information you want to edit.​

Step 3: Save​

Select [Save] to update assignment.

3. Delete Assignment​

Click the [Edit] button at the right corner of the assignment.

Select [Delete] to delete the assignment.

3. Grade the assignment.​

Step 1: Select the assigned assignment.​

Press the [View] icon in the Action section to view and grade the assignment.

The system will display detailed information about the assigned assignment.

Step 2: View Submissions​

Click [Student Submissions] on the toolbar to check the submissions of students.

The system displays a list of students who have submitted assignments and students who have been graded.

Step 3: Select Student​

Click on the name of the student you need to check the assignment for.

The system displays the student's assignment information with actions to View, Delete, and Download the assignment.

Click [History] to view the submission history of the student.

Step 4: Grading​

After reviewing the submitted assignment, the instructor proceeds to grade it and return it to the student.

4. Chat with students​

Select the student you want to discuss with directly, then click [Chat] to start a conversation with the student.

The system will display the chat box and chat history between you and the student. Here, you can interact, discuss, provide support, and guide the student.
